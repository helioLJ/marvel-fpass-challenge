## Stack


**ReactJS, TypeScript e Styled Components**


## Running Locally

Clone

```bash
  git clone https://github.com/helioLJ/marvel-fpass-challenge
```

Change to directory folder

```bash
  cd marvel-fpass-challenge
```

Install dependencies

```bash
  npm install
```

Run the server

```bash
  npm run dev
```
